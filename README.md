# Profil Sekolah SD Negeri 1 Jakarta

Website profil sekolah statis yang modern, responsif, dan profesional.

## Fitur
- Beranda dengan Hero Section
- Profil Sekolah (Sejarah, Visi Misi)
- Daftar Guru & Staff
- Galeri Kegiatan (Lightbox)
- Informasi Kontak & Peta

## Teknologi
- HTML5
- CSS3 (Custom Modern Design)
- JavaScript (Vanilla)
